# MindEase AI — UI/UX Case Study

MindEase AI is a wellness-focused mobile application concept designed to support emotional wellbeing through mood tracking, AI-assisted conversations, mindfulness activities, and personalized wellness insights.

## Features
- Mood Tracking
- AI Wellness Support
- Emotional Insights
- Wellness Dashboard
- Calm & Minimal UI Design

## Tools Used
- Figma
- UI/UX Design
- Prototyping

## Screens Included
- Onboarding
- Login
- Home Dashboard
- AI Chat
- Insights & Analytics
